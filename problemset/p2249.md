## Problem
Given `circles` where `circles[i] = [xi, yi, ri]`, return the number of lattice points inside at least one circle.

**Notes**
- Lattice points have integer coordinates.
- Points on the circumference are included.

## Examples
**Input:** `circles = [[2,2,1]]`  
**Output:** `5`  
*Explanation:* Valid points: `(1, 2), (2, 1), (2, 2), (2, 3), (3, 2)`

**Input:** `circles = [[2,2,2],[3,4,1]]`  
**Output:** `16`  
*Explanation:* Exactly 16 points fall within the circles (e.g., `(0, 2), (2, 0), (2, 4), (3, 2), (4, 4)`)

## Constraints
- `1 <= circles.length <= 200`
- `circles[i].length == 3`
- `1 <= xi, yi <= 100`
- `1 <= ri <= min(xi, yi)`
