## Problem
Given `rectangles` (`[li, hi]`) and `points` (`[xj, yj]`), return an array `count` where `count[j]` is the number of rectangles containing `points[j]`.
- Rectangle `i` spans from `(0, 0)` to `(li, hi)`.
- Contains a point if `0 <= xj <= li` and `0 <= yj <= hi` (edges included).

## Examples
**Input:** `rectangles = [[1,2],[2,3],[2,5]], points = [[2,1],[1,4]]`  
**Output:** `[2, 1]`  
*Explanation:* `(2,1)` is in 2 rectangles; `(1,4)` is in 1.

**Input:** `rectangles = [[1,1],[2,2],[3,3]], points = [[1,3],[1,1]]`  
**Output:** `[1, 3]`  
*Explanation:* `(1,3)` is in 1 rectangle; `(1,1)` is in 3.

## Constraints
- `1 <= rectangles.length, points.length <= 5 * 10^4`
- `rectangles[i].length == points[j].length == 2`
- `1 <= li, xj <= 10^9`
- `1 <= hi, yj <= 100`
- All rectangles and points are unique.
